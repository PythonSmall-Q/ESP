# user-service (占位)

技术栈: Java + Spring Boot 3 + Spring Security + JPA + PostgreSQL

占位说明：此目录目前仅包含服务说明文件。后续将添加 `pom.xml`、`Application.java`、`Dockerfile`、`src/main/...` 等完整实现。

API 概述：
- POST /api/v1/auth/login
- POST /api/v1/auth/refresh
- GET  /api/v1/users
- POST /api/v1/users/{id}/roles
- GET  /api/v1/permissions/check
